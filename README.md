# stickman-game
# 🎮 Stickman: Аркада

Простая, но динамичная браузерная игра, в которой ты управляешь палочным человечком, стреляешь по врагам, собираешь бонусы и проходишь уровни.

## 🕹️ Управление
- `W A S D` или `Стрелки` — движение
- `Пробел` — выстрел

## ⚡ Механики
- Враги падают сверху — не дай им коснуться тебя!
- Каждые 20 очков — новый уровень, враги становятся быстрее
- Зелёные бонусы: +10 очков и +1 жизнь
- У тебя 5 жизней. Не проиграй!

## 🔊 Звуки
Встроенные звуки через Web Audio API — выстрелы, попадания, уровень ап.

## 🌐 Где играть
Игра работает прямо в браузере. Открой `index.html` и играй!
